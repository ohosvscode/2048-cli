# 隐私政策

**生效日期：** 2025年12月28日

**更新日期：** 2025年12月28日

2048 CLI（以下简称"本应用"）尊重并保护所有使用本应用用户的个人隐私权。为了给您提供更准确、更优质的服务，本应用会按照本隐私政策的说明收集、使用、存储和保护您的信息。请您仔细阅读本隐私政策，在确认充分理解并同意后使用本应用。如果您不同意本隐私政策的内容，将可能导致本应用无法正常运行，或者无法达到我们拟达到的服务效果，您应当立即停止访问/使用本应用。

## 一、适用范围

本隐私政策适用于 2048 CLI 应用及其提供的所有功能和服务。

## 二、信息收集与使用

### 2.1 本应用收集的信息类型

#### 2.1.1 设备信息
- **设备型号**：用于适配不同设备的功能和性能
- **操作系统版本**：用于确保应用兼容性和稳定性
- **设备标识符**：用于应用功能正常运行

#### 2.1.2 应用使用信息
- **应用运行日志**：用于诊断和解决技术问题，提升应用稳定性
- **性能数据**：用于优化应用性能

#### 2.1.3 网络信息
- **网络连接状态**：用于提供网络服务功能
- **IP地址**：当您使用网络功能时，系统会自动记录您的IP地址

### 2.2 用户存储的数据

本应用作为 2048 CLI 数据库服务器，您可以通过本应用存储、管理和处理数据。这些数据包括但不限于：

- **键值对数据**：您通过 2048 CLI 命令存储的各种类型的数据
- **配置信息**：2048 CLI 服务器的配置参数
- **日志文件**：2048 CLI 服务器运行产生的日志

**重要提示**：
- 上述数据完全存储在您的设备本地，本应用不会上传、分享或传输这些数据到任何第三方服务器
- 您对存储在本应用中的数据拥有完全的控制权
- 您可以通过 2048 CLI 命令随时查看、修改或删除您存储的数据

### 2.3 信息使用目的

我们收集上述信息仅用于以下目的：

1. **提供服务**：确保 2048 CLI 服务器功能正常运行
2. **改进服务**：分析应用使用情况，优化应用性能和用户体验
3. **技术支持**：诊断和解决技术问题
4. **安全保障**：检测、预防和解决安全威胁

## 三、信息存储

### 3.1 存储位置

- **本地存储**：所有 2048 CLI 数据文件、配置文件、日志文件均存储在您的设备本地存储空间中
- **存储期限**：数据将一直保存在您的设备上，直到您主动删除应用或清除数据

### 3.2 数据安全

我们采用行业标准的安全措施保护您的信息：

- **本地加密**：敏感数据在存储时采用加密技术
- **访问控制**：通过系统权限机制控制对数据的访问
- **安全传输**：如使用网络功能，数据传输采用安全协议

## 四、信息共享与披露

### 4.1 信息共享原则

我们不会向任何第三方出售、交易或转让您的个人信息。仅在以下情况下，我们可能会共享您的信息：

1. **法律法规要求**：根据法律法规、法院判决、政府机关的要求
2. **紧急情况**：为保护用户或公众的生命、财产等重大合法权益

### 4.2 第三方服务

本应用不包含任何第三方 SDK，不会与任何第三方服务共享您的信息。

## 五、您的权利

根据相关法律法规，您对自己的个人信息享有以下权利：

1. **访问权**：您有权访问本应用存储的您的数据
2. **更正权**：您有权更正不准确的数据
3. **删除权**：您有权删除您存储的数据
4. **撤回同意**：您可以通过卸载应用撤回对本隐私政策的同意

## 六、未成年人保护

我们非常重视对未成年人个人信息的保护。如果您是18周岁以下的未成年人，建议您请您的父母或监护人仔细阅读本隐私政策，并在征得您的父母或监护人同意后使用本应用。

## 七、隐私政策的变更

我们可能会适时对本隐私政策进行修订。当隐私政策发生变更时，我们会在应用内以显著方式提醒您相关内容的更新。请您定期查看本隐私政策，以了解最新的隐私保护措施。
